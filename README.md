# FUTURE_DS_02
# 📉 Customer Retention & Churn Analysis

## Project Overview

Customer churn is one of the most critical challenges faced by subscription-based businesses. This project uses Power BI to analyze customer retention patterns and identify the factors contributing to customer churn.

The dashboard provides detailed insights into customer demographics, contract types, internet services, payment methods, and support services to help businesses improve customer retention strategies.

---

## 🎯 Objectives

* Analyze customer churn behavior.
* Identify factors influencing customer retention.
* Understand customer demographics and service usage.
* Discover high-risk customer segments.
* Support data-driven retention strategies.

---

## 🛠️ Tools & Technologies

* Power BI Desktop
* Power Query
* DAX (Data Analysis Expressions)
* CSV Dataset

---

## 📊 Dashboard Features

### Churn Analysis

* Churn Distribution by Tenure Group
* Churn Analysis by Contract Type
* Churn Analysis by Gender
* Churn Analysis by Internet Service

### Customer Segmentation

* Contract Type Analysis
* Payment Method Analysis
* Partner & Dependents Analysis
* Technical Support Analysis

### Advanced Visualizations

* Decomposition Tree for Churn Drivers
* Interactive Slicers and Filters
* Customer Segmentation Analysis

---

## 🔍 Key Insights

* Customers with month-to-month contracts showed significantly higher churn rates.
* Lack of technical support was strongly associated with customer churn.
* Customers using fiber optic internet services experienced higher churn levels.
* Electronic check payment users were more likely to churn.
* Long-term contract customers demonstrated stronger retention.
